Dit is een code dat is aangemaakt voor de toevoeging van een industriële CAT-camera (MER2-041-302GC-P) aan de Knikker Cel van het bedrijf Hoppenbrouwers.

De code bevat de Galaxy View SDK van de uitgever Daheng Imaging.
Hierbij is de voorbeeldcode "GxChunkdata.cpp" van de uitgevers aangepast zodat de RGB-waarden uitgelezen kunnen worden en het gebied dat de camera uitleest.

Ook bevat de code libraries van de uitgever Stéphane Raimbault  om de modbus tcp configuratie te laten werken. Met behulp van de combinatie hiervan is het mogelijk om de waarden van de camera uit te lezen en door te sturen naar een PLC-programma zoals Siemens of Codesys.
